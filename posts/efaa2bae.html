¶1.行测中的类别分析

¶1.1资料分析

¶1.1.1 统计术语

  • 基期、现期

    • 作为对比参照的时期称为基期，而相对于基期的为现期

      例如：今年比去年公司营收增加100万。 基期：去年 现期：今年

  • 增长量

    • 增长量是指基期量与现期量增长（或减少）的绝对量
    • 表述特征为：…比…增长（下降）某个具体值

    增长量=现期量−基期量{\color{Purple} \mathbf{增长量 = 现期量 - 基期量} }增长量=现期量−基期量

  • 增长率（rate）

    • 是指增长量与基期量的相对变化，即在基期量的基础上增长了多大的幅度
    • 增长率又称为增幅、增速、增长幅度、增长速度等

    增长率（%）=增长量基期量{\color{Purple} \mathbf{增长率（\%） = \frac{增长量}{基期量} } }增长率（%）=基期量增长量​

    • 例如：去年营收为100万今年为130万，增长率是多少

      r=130−100100=30%{\color{Purple} \mathbf{r = \frac{130-100}{100}=30\% } }r=100130−100​=30%

      去年产值：

      去年产值=现期量1+增速{\color{Purple} \mathbf{去年产值 = \frac{现期量}{1+增速} } }去年产值=1+增速现期量​

    • 年均增长量：

    去年增长量=现期量−基期量间隔年份{\color{Purple} \mathbf{去年增长量 = \frac{现期量-基期量}{间隔年份} } }去年增长量=间隔年份现期量−基期量​

    • 年均增长率：

      年均增长率=基期量×(1+年均增长率)n{\color{Purple} \mathbf{年均增长率= 基期量\times\left ( 1+ 年均增长率 \right )^n } }年均增长率=基期量×(1+年均增长率)n

      n为相差年份

  • 同比

    • 与历史同期相比较

  • 环比

    • 现在统计周期和上一个统计周期相比较

  • 比重

    • 是指某部分在总体中所占的百分比，一般是百分数形式

    比重=部分整体{\color{Purple} \mathbf{比重=\frac{部分}{整体} } }比重=整体部分​

  • 倍数

  • 平均数

  • 成数

  • 翻番

    • 翻一番为原来的2倍

  • 顺差

  • 逆差

  • GDP

    • 第一产业增加值+第二产业增加值+第三产业增加值，增加值就是问GDP

  • 恩格尔系数

    • 食品支出总额占个人消费支出总额的比重，比例越低生活水平越高

  • 基尼系数

    • 衡量一个国家或地区居民收入差据的常用指标，系数越大，越不平等

  • 五年计划

    • 13五
